## ❐ 401. Binary Watch

|      |                                                        |
|------|--------------------------------------------------------|

<br>

### 핵심

문제의 핵심은 조합(Combination)을 찾는 것이다.

<br>

### 풀이 순서

1. hour 조합과 min 조합을 찾는다.
    - hour의 경우 조합의 각 요소 총 합이 12를 초과해서는 안된다.
    - min의 경우 각 요소 총 합이 60을 초과해서는 안된다.
2. 찾은 조합들을 "hour:min" 형식으로 변형한 후 결과 배열에 삽입한다.
